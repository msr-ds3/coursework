Question 5.20:

Data: mean shoulder girth = 108.20 cm, sd shoulder girth = 10.37 cm, mean height = 171.14 cm, sd height = 9.41 cm, corr = R = 0.67,

  1. Write the equation of the regression line for predicting height.
  • b1 = (9.41/10.37) * 0.67 = 0.60797
  • h = height, s = shoulder girth
  • h - 171.14 = b1(s - 108.20)
  • happrox = 105.36 + 0.608(s)
  1. Interpret the slope and intercept in this context.
  • The slope over here means that, for each increase of 1 cm in shoulder girth, height is expected to increase by 0.608 cm.
  • The intercept shows the average height to be 105.36 if shoulder girth were zero.
  1. Calculate R2 of the regression line for predicting height from shoulder girth, and interpret it in the context of the application.
  • R2 = 0.672 = 0.4489. About 45% of variability is accounted for by the model.
  1. A randomly selected student has a shoulder girth of 100 cm. Predict the height of this student using the model.
  • happrox = 105.36 + 0.608(100) = 166.16
  1. The student from (d) is 160 cm tall. Calculate the residual and explain what this residual means.
  • Residual = e = 160 - 166.6 = -6.6. The difference is negative so the model slightly overestimated but the expected is still close to the estimated.
  1. A one year old has a shoulder girth of 56 cm. Would it be appropriate to use this linear model to predict the height of this child?
  • The sample was taken from an older population so it would not be appropriate to use to estimate the heights of children.

Question 5.29

  1. Describe the relationship between height and weight.
  • There is a positive, linear relationship between height and weight. (As height increases, weight increases.)
  1. Write the equation of the regression line. Interpret the slope and intercept in context.
  • h = height, w = weight
  • wapprox = -105.0113 + 1.0176(h)
  • The slope here means that, for every increase of 1 cm in height, the model predicts an increase of 1.0176 kg in weight.
  • The Intercept shows the average weight of someone who is 0 cm tall to be -105.0113 kg.
  1. Do the data provide strong evidence that an increase in height is associated with an increase in weight? State the null and alternative hypothesis, report the p-value, and state your conclusion.
  • H0: beta1 = 0
  • HA: beta1 != 0
  • The p-value is 0.0000 so it is very small and therefore very unlikely for these results to occur given the null hypothesis being true so we reject the null hypothesis here and conclude that there is a relationship between height and weight.
  1. The correlation coefficient for height and weight is 0.72. Calculate R2 and interpret it in context.
  • R2 = 0.722 = 0.5184
  • About 52% of the variability in weight is accounted for by the model.
